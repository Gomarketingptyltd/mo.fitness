# Mo.fitness Website Content List

The website design is fixed.
Please provide the following content for replacement.

## 1. Images

### Logo `optional`
- Position: top left of the website
- Preferred format: transparent `PNG` or `SVG`

### Main Hero Image `recommended`
- Position: first screen, right side
- Preferred type:
  - gym interior
  - coach portrait
  - training floor
  - front desk / entrance / brand wall
- Preferred format: high-resolution `JPG` or `PNG`

## 2. Brand Information
- Brand / business name
- Website title
- Short website description for Google / link preview
- Website handle or sub-label if needed

## 3. Hero Section
- Top short line above the headline
- Main headline line 1
- Main headline line 2
- Short introduction paragraph
- Primary button text
- Secondary button text
- 3 short stats or key selling points

## 4. Facility / Hero Visual Section
- Small section label
- One short headline
- 4 short feature blocks
- 2 location / category tags
- One short gym description
- 3 quick highlights
- 1 intro offer line

## 5. About Section
- Section title
- Small heading
- 2 short paragraphs
- 3 small highlight blocks
- 4 bullet points about what members care about

## 6. Services / Training Section
Please provide content for 3 services.

For each service:
- Service name
- One short description
- 3 bullet points

Current service structure:
- Open Gym Access
- Coached Strength Classes
- Small Group Coaching

## 7. Timetable Section
- Section title
- One short intro paragraph
- 3 timetable rows

## 8. Testimonials
Please provide 2 testimonials.

For each testimonial:
- Quote
- Name
- Role / title `optional`

## 9. Membership / Pricing Section
Please provide content for 3 membership options.

For each option:
- Plan name
- Price
- 3 bullet points
- Button text

Current pricing structure:
- 7-Day Pass
- Unlimited Club
- Small Group Coaching

## 10. Contact Section
- Phone number
- Email address
- Instagram handle
- Address
- Opening hours
- Parking / transport note

## 11. Contact Form `optional`
- Form title
- Form description
- Submit button text

## 12. Simple Submission Format

Please send the content in this format:

```text
1. Brand name:
2. Website title:
3. Short website description:
4. Hero headline:
5. Hero image:
6. 3 services:
7. 3 pricing options:
8. Timetable:
```
